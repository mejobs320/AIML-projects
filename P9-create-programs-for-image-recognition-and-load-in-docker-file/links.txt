docker file create and run

__To build and run the container:__

docker --version

Open a terminal in the `P9-Docker` folder:
docker build -t ai-image-classifier .

Run the container:
docker run -p 8501:8501 ai-image-classifier
  You can now view your Streamlit app in your browser.
  Local URL: http://localhost:8501


useful commands:
Run in background (detached mode):
powershelldocker run -d -p 8501:8501 ai-image-classifier
See running containers:
docker ps   (to know contain id)
Stop a background container:
docker stop <container_id>
