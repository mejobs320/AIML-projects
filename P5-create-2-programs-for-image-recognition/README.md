# programs-for-image-recognition.
Create image recognition with groq llm  (used llm - Llama Vision)  Model:** `llama-3.2-11b-vision`. use program main.py

Use groq llm  (used llm - Llama Vision)  Model:** `llama-3.2-11b-vision`            ----- main.py  with streamlit
streamlit will be used to create webui
a web ui will open. you will share image in web ui and llm (llama-3.2-11b-vision) will recognize image and share details in web ui

command:
streamlit run main.py   

Result:
llm (llama-3.2-11b-vision) will recognize image and share details in web ui interface
